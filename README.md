# ML-проект для предсказания стоимости жилой недвижимости.

## Цель проекта
Создание end-to-end решения для предсказания стоимости домов на основе их характеристик. Проект реализуется полностью с нуля, без использования готовых решений.

### Основные задачи:
- Сбор и генерация реалистичных данных о недвижимости

- Полный цикл EDA (Exploratory Data Analysis)

- Реализация пайплайна предобработки данных

- Разработка и сравнение нескольких ML-моделей

- Создание веб-интерфейса для инференса

- Документирование каждого этапа

### Стек технологий:

- Python 3.9+

- Data Science Stack: pandas, numpy, scipy

- Визуализация: matplotlib, seaborn, plotly

- Машинное обучение: scikit-learn, xgboost

- Веб-фреймворк: Flask

- Разработка: PyCharm

- Версионирование: Git
